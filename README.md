## Install:

### Comands:
    yarn add express
    yarn add typescript
    yarn add ts-node-dev -D
    yarn add @types/express -D
    yarn add uuidv4
    yarn add dotenv
    yarn add knex
    yarn add mysql

### teste:
    yarn start
    Rota de teste: localhost:3333/test
    Essa rota deve retornar um json {"sucess":"true"} e um http status 200
